# 药

## 药的产生

每次去医院看病，我们都要做一堆检查。当医生说“没什么事，回家多休息就好”，或是“上火了，吃点药下下火”时，我们会瞬间松一口气。可一旦听到“得做手术”，心情便直接跌到谷底。
其实，医学和数学很不一样。数学依赖严密的逻辑推理得出结论，医学则非常看重经验。医生的话并不像“1+1=2”那样确定，很多时候是基于经验的判断——而经验也可能出错。听到这儿，你是否会倒吸一口凉气？如果医生对你说：“这个问题大概有80%的概率需要动手术”，你又会怎么想呢？

在古代，医生看病主要靠“问诊”。那时人们对人体的构造和功能了解甚少，更难以弄清疾病的真正成因。“望闻问切”是我们常听说的古老智慧，可治疗过程却有点像面对一个“黑盒”：医生给病人用药或做手术，再观察表面症状是好转还是恶化。至于身体内部发生了什么变化，基本看不出来。
如果一种药总吃死人，就会被禁用；如果一种药一吃就见效，就会被归为有效药物。经验就这样一点点积累下来。可实际上，自然界大部分药物的效果并不那么明显，对有些人有效，对另一些人却没用。光靠简单观察来给药物分类，作用毕竟有限。

随着经验不断积累，东方与西方都涌现出一批影响深远的药典，例如中国的《神农本草经》《本草纲目》，古希腊的草药志、中世纪的修道院药典，以及迪奥斯科里德斯的《药物论》。此外，古埃及的《埃伯斯纸草文稿》、古罗马盖伦的《医学艺术》，以及11世纪波斯学者伊本·西那的《治疗之书》，也都是医学史上的重要著作。
古埃及的文明以医学水平见长。然而他们留下的260多个药方，如今经过验证，只有四分之一含有真正有效的药物成分。这些方子被使用了上千年，不可能全无效果，但其中许多所谓的“疗效”，恐怕更多是心理作用，而非药物成分带来的。


## 药的发展

药的发展：人类与疾病的一场长期博弈

上一节我们聊过，药是怎么产生的。如果用一句话概括药物史，那就是——人类与疾病的一场长期博弈。人类并不是一开始就“懂得如何治病”，而是在漫长的失败、试错与顿悟中摸索，这个过程可以分为三次跃迁。


第一跃迁：先做点什么
把时间拨回上古。一个人突然病倒了——发烧、疼痛、抽搐。他不知道什么是病毒、细菌、免疫系统，只知道一件事：如果什么都不做，人可能会死。
而人类，无法接受“毫无原因的痛苦”。哪怕解释是错的，也好过完全没有解释。

于是，疾病被理解为——神灵的惩罚、鬼怪的附身、阴阳失衡、诅咒降临。既然病因来自“看不见的力量”，治疗方式自然也是仪式化的：祈祷、符咒、献祭、驱邪。

但慢慢地，人们发现了一件重要的事：有些时候，不靠神，人也会好转。在恐惧与无助中，开始尝试“自救”——割掉腐烂的组织，敷上某些植物，煮汤、涂抹、吞服。
“先做点什么”，成了医学的最初起点。

很快，人们注意到一些奇怪的现象：有人吃了某种叶子，疼痛减轻了；有人喝了某种汤，发烧退了；但也有人吃错了，直接死了。
于是，一套朴素却残酷的规则诞生了：吃了好 → 记下来,吃了死 → 传为禁忌

经验开始被一代代积累，形成了早期医学典籍：中国有《本草纲目》，西方有《药物论》《治疗之书》。这，就是最原始的草药医学。它像一个巨大的“经验数据库”。
但有一个致命缺陷：它不知道为什么有效。剂量不稳定、效果不可预测、对不同的人结果差异巨大。很遗憾的是，今天所有的能找到的植物，都被制药公司翻了个遍，植物中有效的化合物都被一一尝试。这些古书上的绝大部分植物都和对应的疾病没有什么关系。

那些看起来被治好了，是来自于人体自身的免疫力和安慰剂效应。
即使今天，许多草药学，本质上仍停留在这个阶段。


### 第二次跃迁：从“整株草”到“有效部分”

大部分植物的有效成分含量太低，都不能启到立竿见影的效果。绝大多数植物的有效成分太少，
随后，一次关键的转变发生了。

人们开始追问：到底是什么在起作用？

真正的突破在于一句话：不是草有用，是草里的“某个东西”有用。

随着蒸馏、煎煮、酒精萃取等技术出现，人类第一次可以“拆解”草药。

结果令人震惊：同一种植物——根有用，叶没用；
成分不同，效果完全不同；剂量，终于开始变得稳定。

📌 经典例子：
柳树皮 → 解热镇痛 → 后来的阿司匹林
罂粟汁 → 镇痛、镇静

👉 这一步，药理学真正开始了。


### 第三次跃迁：化学革命——“我们看见了分子”

真正的分水岭，出现在 18–19 世纪。

化学的发展，让人类第一次具备了关键能力：分离、纯化、结晶、写出分子式。

标志性事件接连发生：1805 年：吗啡从罂粟中被成功分离
奎宁从金鸡纳树中提取
咖啡因、尼古丁相继被发现

这是第一次——药物拥有确定的成分、确定的剂量、可重复的效果。

也正是在这一刻，传统经验医学被正式甩在了身后。



### 药理学诞生：从“有用吗”到“为什么有用”

接下来，人们不再满足于“能治病”，而开始问：它怎么起作用？
作用在哪里？为什么会产生这些效果？受体理论、神经递质、酶抑制等概念陆续出现。

📌 例如阿司匹林：
它通过不可逆抑制 COX 酶，减少前列腺素和血栓素的生成，
从而产生解热、镇痛、抗炎和抗血小板作用。

药，第一次真正“被理解了”。


#### 从“发现”到“设计”：现代药物的诞生

更深层的变化随之发生——人类不再被动等待自然给答案，而是主动设计分子。

典型路径是：天然分子 → 改造 → 人工合成新分子

目标只有三个：更强效、更安全、更稳定。

📌 例子：
青蒿素 → 双氢青蒿素
吗啡 → 芬太尼（危险，但极具代表性）

### 最后一道门槛：循证医学与临床试验

真正决定“能不能成为药”的，是最后一步。

四条决定性规则

随机： 让分组完全交给概率，为什么必须随机？, 如果不随机，人会不自觉地：把病情重的放一组, 把年轻的放一组,把“看起来更有希望的”放一组
这是唯一能同时对付“已知偏倚 + 未知偏倚” 的方法。

双盲： 病人不知道吃的是什么，医生也不知道。
因为“知道本身”就会改变结果。为什么“知道”本身就是问题？ 病人知道：觉得“我吃的是好药” 主观症状改善
   医生知道：更耐心更积极评估疗效， 防止心理作用偏差。 生活类比，评委不知道谁是自己学生

对照： 实验组，  新药 对照组  安慰剂 / 旧药 / 标准治疗,
区分三种效应：自然好转、安慰剂效应、药物真实作用。

大样本, 为什么样本一定要大？ 人少时，运气太大；人多时，规律才会显现。
统计学的本质，是压制随机噪声。 例如随机抛硬币 2 次，都是正面的概率很大。

这一关，彻底切断了“经验即真理”。


## 尾声：不是传统输了，是科学赢了

许多传统草药，在这一关被淘汰；
少数幸存下来，成为了现代药物。

这并不是对传统的否定，
而是一次残酷但必要的筛选。

最终留下来的，不是信仰，
不是经验，
而是——可被验证的真理。

一、草药阶段 0：生病了总得做点什么，不能坐以待毙。
草药医学的原始形态（经验有效，但不知为什么）
核心特征，靠经验：
吃了好 → 记下来
吃了死 → 传为禁忌


👉 这是“经验数据库”，不是科学

二、阶段 1：经验开始“去神秘化”（但还没科学）
关键变化， 开始否定明显无效或迷信成分
强调“反复可用”

无对照组

无剂量控制

无机制解释

不知道有效成分,剂量极不稳定, 中西都一样, 参考前面的视频一，他很不准确。

中国：《神农本草经》《本草纲目》
西方：希腊草药志、修道院药典


例子：
李时珍批评“某物避邪”之说， 西方草药学去掉占星、咒语
👉 这是理性态度，但还没有实验。

三、阶段 2：从“整株草” → “有效部分”（方法论第一次跃迁）
核心突破
不是草有用，是草里的“某个东西”有用

技术条件
蒸馏，煎煮， 酒精萃取

结果， 同一种草：
根有用，叶无用， 剂量开始稳定

📌 历史例子：
柳树皮 → 解热镇痛（后来变成阿司匹林）
罂粟汁 → 镇痛、镇静， 👉 这是药理学的真正起点。


四、阶段 3：化学革命——“看见分子”
（18–19 世纪，决定性阶段）
关键能力，分离，纯化，结晶

分子式

标志性事件: 1805：吗啡从罂粟中被分离, 奎宁从金鸡纳树中提取, 咖啡因、尼古丁相继被发现

👉 第一次：
有确定成分，有确定剂量，有可重复效果

📌 这一步，中医与传统草药被正式“甩开”




比如——
吃韭菜，壮阳，
因为它长得快、生命力强。

吃腰子，补肾，
因为肾长得像肾，
这叫“以形补形”。

老虎很强，
所以虎骨能治风湿，
虎皮能保命、驱邪。

猴子看起来聪明，
那吃猴脑，
是不是也能让人变聪明？

川三甲会打洞、会穿行，
那它一定“通”，
吃了就能疏通血管。

今天我们当然知道，
这些做法没有科学依据。

但问题是——
为什么这些观念，能流传几千年？
