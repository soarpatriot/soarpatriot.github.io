# 药


## 药的发展

药的发展：人类与疾病的一场长期博弈

上一节我们聊过，药是怎么产生的。如果用一句话概括药物史，那就是——人类与疾病的一场长期博弈。人类并不是一开始就“懂得如何治病”，而是在漫长的失败、试错与顿悟中摸索，归纳为以下几个阶段


1. 史前及新石器时代：纯粹的运气与试错
如果用一句话概括药物史，
那就是——
人类与疾病的一场长期博弈。

但人类，并不是一开始就懂得怎么治病。
恰恰相反，
最早的医学，几乎全靠——
运气。

把时间拨回史前和新石器时代。
那时的人类，对身体一无所知。
不知道器官，不知道细菌，
甚至不知道“病”是怎么来的。

但人类有一个无法改变的本能：
我们无法接受毫无原因的痛苦。

于是，各种解释诞生了。
有人相信，疾病是神灵的惩罚；
有人认为，是鬼怪附身、诅咒降临；
也有人觉得，是阴阳失衡、气运出了问题。

既然病因来自“看不见的力量”，
治疗方式自然也是仪式化的：
祈祷、符咒、献祭、驱邪。

但与此同时，
人类也开始偷偷做另一件事——
自救。

他们咀嚼树根、树叶，
把植物煮汤、涂抹、吞服。
希望它们能减轻疼痛，
同时也祈祷——
自己不要先被毒死。

在纯粹的试错中，
一些“有效”的东西被偶然留下来：
鸦片、酒精、蛇根草、杜松、乳香、茴香、桦木菌……
不是因为理解了原理，
只是因为——
吃了没死，甚至还好了点。

慢慢地，人们发现一件重要的事：
有时候，
不靠神，人也会好转。

于是，医学最早的一条原则诞生了：

先做点什么。

割掉腐烂的组织，
敷上某些植物，
哪怕只是“试一试”。

很快，一套残酷却高效的规则形成了：
吃了有用 → 记下来；
吃了会死 → 传为禁忌。

而这种阶段留下的“直觉医学”，
一直影响到今天。

比如——
吃韭菜，壮阳，
因为它长得快、生命力强。

吃腰子，补肾，
因为肾长得像肾，
这叫“以形补形”。

老虎很强，
所以虎骨能治风湿，
虎皮能保命、驱邪。

猴子看起来聪明，
那吃猴脑，
是不是也能让人变聪明？

川三甲会打洞、会穿行，
那它一定“通”，
吃了就能疏通血管。

今天我们当然知道，
这些做法没有科学依据。

但问题是——
为什么这些观念，能流传几千年？

2. 植物时代
植物时代：人类第一次真正“积累经验”

在经历了纯靠运气的试错之后，
人类进入了药物史的第二个阶段——
植物时代。

这一阶段，
植物几乎被视为神留下的药典。

花、根、种子、树皮、树汁、
苔藓、真菌、海藻……
植物的每一部分，都可能是药。

人类开始系统地采摘、去皮、研磨、晒干、烧煮，
把自然，
一点点变成可以入口的“良药”。

直到今天，
这些方法依然随处可见。
煎、煮、泡、敷、熬、涂——
我们对“药”的直觉，
很多就来自这个时代。

和之前最大的不同在于：
经验，开始被保存下来。

不再只是口耳相传，
而是被一代一代地记录、整理、累积。

于是，早期医学典籍诞生了：
中国有《本草纲目》，
西方有《药物论》《治疗之书》。

它们本质上，是人类最早的——
“经验数据库”。

吃了什么，
对什么病，
有没有效果，
有没有毒。
全部写下来。

但这个体系，有一个致命的缺陷。

它知道——“有没有用”，
却不知道——“为什么有用”。

同一种植物，
不同产地、不同季节、不同部位，
有效成分天差地别。

剂量不稳定，
效果不可预测，
对不同的人，结果差异巨大。

更残酷的是，
站在今天回头看，
人类能找到的植物，
几乎已经被制药公司翻了个遍。

植物中可能有效的化合物，
被一一提取、分离、测试。

结论并不浪漫：
古书里记载的绝大多数植物，
和对应的疾病，
并没有直接关系。

绝大多数植物的有效成分含量都很低，没有起到治疗的效果，很多人现在还迷恋“纯天然”，甚至觉得“野生的更好”。
像一些常见的植物，罂粟，金鸡纳树皮，五倍子、丁香，黄连、甘草、人参，麻黄、青蒿、银杏，
等一些成分含量高的植物才能起效。这些药，时常会被搭配组合其它植物一起治病。成分被煎煮破坏 / 稀释，或产生化学反应，
以及不同的医生开出的“千人千方”，最后喝到肚里的是什么，很难说的清楚了。 我们知道只有有效的成分才能起作用，那些不起作用的辅助药物，有时候反而会干扰，甚至使有效成分失效。
在不能定量分析的时候，好多事情就说不清楚了。
病人吃了会感觉到起效，吃药后有感觉，却并不一定能治疗对应的病。

那种长期服用的，副作用是会积累的。看如下表格。

那些“被治好”的案例，
很多时候，
来自人体自身的免疫力，
以及一个被严重低估的力量——
安慰剂效应。

甚至直到今天，
相当一部分草药学，
本质上仍停留在这个阶段：
有效，但不稳定；
被相信，却说不清原因。

这也正是医学接下来必须跨过去的一道坎——
从经验，走向理解。

3. 工业时代
无知是激励人们提出问题，追求进步的重要动力，化学是从中世纪的炼金术发展而来，最初的炼金术相信知识早已存在，只是被人藏匿起来了，直到18世纪中叶，化学才摆脱这些妄想，作为一种有价值的科学走向世界。
从发现甜菜里面的糖结晶，人们认识到一种分子，无所谓来自何处，其自身的性质才是关键。随着实际商业利益的的需求，判断药物是否被稀释和掺假，从甜菜中提取糖变的有利可图，这促使化学进一步发展，精确测量成分，溶剂萃取，分离提纯。

植物的特质随季节，气候，泥土而变化，难以从中获得稳定一致的药品。 化学技术可以把植物中治病的有效成分提取，而避免有其中有有害成分。人们也不用去吃各种混合蒸煮成分不明的混合物。上一节我们说到，许多植物的副作用。现在依然有人认为植物效用温和，而药物是具有刺激性的人造品。那些不合格的药品，以及混入不明杂质的药，是会有其他未预计的副作用。 人工提取的经过数十年验证的药物，副主要是比较明确的。
比如，人工从金鸡纳树皮中多种成分提取出奎宁，这是金鸡纳树皮多种成分之一，具有退烧和抗疟疾的特性。 这要比金鸡纳树皮可口的多，毒性也比较低，也不会令人呕吐。

在可控计量的麻醉剂发明以前，人们使用酒精，罂粟，曼陀罗花等一些列混合物作为手术的麻醉剂，但这些混合物的纯度，无法控制，切除深层皮肤时，疼痛依然无法忍受。
19世纪中叶，手术麻醉剂（如乙醚）的发现让医疗需求爆炸。消毒计和细菌理论的发展，使手术存活率大大提高。
这时候，制药业从手工作坊变成了大工厂。这一时期的核心不在于研发“新”药，而在于利用化工技术为已有药物寻找标准化的生产方式。爱德华·罗宾逊·施贵宝通过建立统一的生产标准，确保了药品质量的一致性，从而催生了大型制药公司。 
化学的发展，让人类第一次具备了关键能力：分离、纯化、结晶、写出分子式。
新撰写的药典非常实用，他包含的条目是化学制品，而不是植物。

标志性事件接连发生：1805 年：吗啡从罂粟中被成功分离
奎宁从金鸡纳树中提取
咖啡因、尼古丁相继被发现

这是第一次——药物拥有确定的成分、确定的剂量、可重复的效果。也正是在这一刻，传统经验医学被正式甩在了身后。


4. 合成时代
原子学说和合成化学的兴起改变了人类对药物的看法——药物不再只是被“发现”，而是可以被**“发明”**。拜耳公司的研究人员发现，通过稍微改变已知分子的化学结构，可以得到效果更好、副作用更小的药物。阿司匹林的诞生便是这一时代的里程碑，它是通过在水杨酸分子上加入乙酰基而合成的,   更深层的变化随之发生——人类不再被动等待自然给答案，而是主动设计分子。

典型路径是：天然分子 → 改造 → 人工合成新分子
目标只有三个：更强效、更安全、更稳定。


5. 魔法子弹
保罗·埃尔利希（Paul Ehrlich）颠覆了传统，提出了**“受体理论”和“有魔力的子弹”的概念。他认为药物可以像钥匙和锁一样，精确地靶向病原体而不伤害人体。他研发的治疗梅毒的洒尔佛散**是历史上第一种完全由人类设计、从零开始研发的合成药物

6. 泥土时代
亚历山大·弗莱明意外发现青霉素后，医药界开启了在泥土中寻宝的时代。塞曼·瓦克斯曼系统性地筛查土壤中的微生物，发现了链霉素。这促使各大制药公司在全球范围内翻挖泥土，寻找新的抗菌化合物

7. 基因技术
基因药物： 随着重组基因技术的突破，人类开始利用大肠杆菌等微生物来生产人类蛋白质药物，如人类胰岛素。
• 理性设计（Rational Design）： 现代猎药师不再仅仅依赖随机筛选，而是基于对受体和酶的生物化学原理的理解，精准地设计分子。例如，卡托普利的研发过程就是通过预测能够抑制ACE酶的化合物结构，仅尝试了几百个分子就获得了成功。


尽管现代技术日益精进，但作者在 sources 中指出，21世纪药物研发的核心挑战依然存在：在不计其数的化合物中反复筛选，且最终成果往往与初始设想大相径庭，运气和毅力在今天依然是猎药师成功的关键因素


### 第二次跃迁：从“整株草”到“有效部分”

大部分植物的有效成分含量太低，都不能启到立竿见影的效果。绝大多数植物的有效成分太少，
随后，一次关键的转变发生了。

人们开始追问：到底是什么在起作用？

真正的突破在于一句话：不是草有用，是草里的“某个东西”有用。

随着蒸馏、煎煮、酒精萃取等技术出现，人类第一次可以“拆解”草药。

结果令人震惊：同一种植物——根有用，叶没用；
成分不同，效果完全不同；剂量，终于开始变得稳定。

📌 经典例子：
柳树皮 → 解热镇痛 → 后来的阿司匹林
罂粟汁 → 镇痛、镇静

👉 这一步，药理学真正开始了。


### 第三次跃迁：化学革命——“我们看见了分子”


### 药理学诞生：从“有用吗”到“为什么有用”

接下来，人们不再满足于“能治病”，而开始问：它怎么起作用？
作用在哪里？为什么会产生这些效果？受体理论、神经递质、酶抑制等概念陆续出现。

📌 例如阿司匹林：
它通过不可逆抑制 COX 酶，减少前列腺素和血栓素的生成，
从而产生解热、镇痛、抗炎和抗血小板作用。

药，第一次真正“被理解了”。


#### 从“发现”到“设计”：现代药物的诞生





### 最后一道门槛：循证医学与临床试验

真正决定“能不能成为药”的，是最后一步。

四条决定性规则

随机： 让分组完全交给概率，为什么必须随机？, 如果不随机，人会不自觉地：把病情重的放一组, 把年轻的放一组,把“看起来更有希望的”放一组
这是唯一能同时对付“已知偏倚 + 未知偏倚” 的方法。

双盲： 病人不知道吃的是什么，医生也不知道。
因为“知道本身”就会改变结果。为什么“知道”本身就是问题？ 病人知道：觉得“我吃的是好药” 主观症状改善
   医生知道：更耐心更积极评估疗效， 防止心理作用偏差。 生活类比，评委不知道谁是自己学生

对照： 实验组，  新药 对照组  安慰剂 / 旧药 / 标准治疗,
区分三种效应：自然好转、安慰剂效应、药物真实作用。

大样本, 为什么样本一定要大？ 人少时，运气太大；人多时，规律才会显现。
统计学的本质，是压制随机噪声。 例如随机抛硬币 2 次，都是正面的概率很大。

这一关，彻底切断了“经验即真理”。


## 尾声：不是传统输了，是科学赢了

许多传统草药，在这一关被淘汰；
少数幸存下来，成为了现代药物。

这并不是对传统的否定，
而是一次残酷但必要的筛选。

最终留下来的，不是信仰，
不是经验，
而是——可被验证的真理。
























随着治疗思想的变化，
于是，人类踏上了一条很漫长的路。这条路，不是从“科学”开始的，而是从恐惧与无助开始的。


第一幕：巫医时代 —— 给未知一个解释

在最早的时候，人们必须解释疾病。

但他们没有显微镜，没有化学，没有统计学，只有直觉。

于是疾病被理解为：

神灵的惩罚

鬼怪的附身

阴阳失衡、诅咒降临

治疗的方式，自然也是仪式、祈祷、符咒、献祭。

📌 这不是愚昧，而是本能。
人类无法接受“毫无原因的痛苦”，哪怕是错的解释，也比没有解释强。

但很快，人们发现了一件事：

有些东西，不靠神，也真的能让人好转。

一、总体演进脉络（先给你一张“路线图”）

不论中西，医学大体都经历了三次范式跃迁：

1. 巫医阶段（神秘解释）
👉 疾病 = 神灵、鬼怪、诅咒、失衡

2. 草药 / 经验医学阶段（经验解释）
👉 疾病 = 可观察、可干预的身体异常

3. 现代医学阶段（科学解释）
👉 疾病 = 生物学机制 + 可验证的因果关系

为什么只到今天，上面 3 个阶段的事情我们仍然都能看到。

在科学史中，很多自然现象相对容易找到因果关系，比如气温差异形成空气对流，产生风，过多的水蒸气蒸发产生雨，地球和太阳之前的万有引力会让地球围绕太阳转动。
但是，和人体相关的现象，背后的原因非常复杂，常常很难搞清楚。比如：
不同人对感冒的反应各异，常得出不同结论：
有人不药而愈，认为吃药无用；
有人只喝热水康复，觉得无需用药；
有人喝姜汤好转，便相信姜汤可治感冒；
有人服药后恢复较慢，反而认为感冒药无效；
有人用药后迅速好转，认定吃药有必要。

比如人得了某种病，吃了某种药好了，是因为药的疗效把病都治好了，还是他吃了药以后产生的心理安慰作用激发身体的免疫力，进而战胜了疾病，抑或是其他什么原因，就很难搞清楚了。而在人的心里更愿意把现实的结果扭曲到自己已经认定好的理论。


一、草药阶段 0：生病了总得做点什么，不能坐以待毙。
草药医学的原始形态（经验有效，但不知为什么）
核心特征，靠经验：
吃了好 → 记下来
吃了死 → 传为禁忌


👉 这是“经验数据库”，不是科学

二、阶段 1：经验开始“去神秘化”（但还没科学）
关键变化， 开始否定明显无效或迷信成分
强调“反复可用”

无对照组

无剂量控制

无机制解释

不知道有效成分,剂量极不稳定, 中西都一样, 参考前面的视频一，他很不准确。

中国：《神农本草经》《本草纲目》
西方：希腊草药志、修道院药典


例子：
李时珍批评“某物避邪”之说， 西方草药学去掉占星、咒语
👉 这是理性态度，但还没有实验。

三、阶段 2：从“整株草” → “有效部分”（方法论第一次跃迁）
核心突破
不是草有用，是草里的“某个东西”有用

技术条件
蒸馏，煎煮， 酒精萃取

结果， 同一种草：
根有用，叶无用， 剂量开始稳定

📌 历史例子：
柳树皮 → 解热镇痛（后来变成阿司匹林）
罂粟汁 → 镇痛、镇静， 👉 这是药理学的真正起点。


四、阶段 3：化学革命——“看见分子”
（18–19 世纪，决定性阶段）
关键能力，分离，纯化，结晶

分子式

标志性事件: 1805：吗啡从罂粟中被分离, 奎宁从金鸡纳树中提取, 咖啡因、尼古丁相继被发现

👉 第一次：
有确定成分，有确定剂量，有可重复效果

📌 这一步，中医与传统草药被正式“甩开”
